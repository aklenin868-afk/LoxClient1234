# LoxClient1234

## Introduction
LoxClient1234 is a versatile client designed for both Forge and Fabric modding environments. This documentation provides all the necessary information to get started with the client, including installation steps, module functionalities, usage instructions, and frequently asked questions.

## Installation

### Forge Installation
1. Download and install the Minecraft Forge installer from the [official Forge website](https://files.minecraftforge.net/).
2. Run the installer and select "Install client".
3. Open the Minecraft launcher and select the Forge profile.
4. Download the LoxClient1234 mod from the releases section of the repository.
5. Place the mod `.jar` file into the `mods` directory located in your Minecraft installation folder.

### Fabric Installation
1. Download the Fabric installer from the [official Fabric website](https://fabricmc.net/use/#installation).
2. Run the installer and select "Install Fabric".
3. Open the Minecraft launcher and use the Fabric profile.
4. Download the LoxClient1234 mod from the releases section of the repository.
5. Place the mod `.jar` file into the `mods` directory located in your Minecraft installation folder.
